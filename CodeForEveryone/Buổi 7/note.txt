điều kiện - conditional
=> giúp cho máy tính thực hiện các logic gì đó dựa vào điều kiện gì đó
if - else
=> Nếu (cái gì đó ) thì (làm gì đó), ngược lại thì (làm gì đó)

=> Nếu trời mưa thì ở nhà, ngược lại thì ra ngoài

trời mưa =  true => ở nhà
false => ra ngoài

Vòng lặp

loop -> thể hiện tính chất lặp đi lặp lại của những hành động / code
- những dòng code lặp đi lặp lại

vòng lặp for
nhận đầu vào 3 biểu thức, mỗi biểu thức cách nhau bằng ;
vòng for cũng phải biểu diễn được số lệnh được thực thi

for (khởi_tạo; điều_kiện; tăng/giảm) {
  // Khối mã thực thi trong mỗi lần lặp
}
